# 物品全卡牌库 (Final Card Catalog - 47/47)

本项目已完成全部 47 张卡牌的逻辑实装与物理验证。

## 一、 核心物理规则
*   **Multiplier (倍率)**: $1 + 污染层数$。
*   **Transmission (传导)**:
    *   **NORMAL**: 沿物品指向方向传播冲击波。
    *   **OMNI**: 向四周所有方向传播冲击波。
    *   **NONE**: 拦截能量流，不向后传递。

## 五、 状态说明
*   **防腐 (Preserved)**: 物品进入不可变状态，污染层数无法通过任何手段增加或减少（由防腐蜡赋予）。
*   **污染结算**: 带有污染的物品在撞击时产生的 $(1+n)$ 次额外循环动作。
