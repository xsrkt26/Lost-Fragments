# 物品系统与全卡牌库 (Card Catalog)

## 一、 物品系统基础
所有卡牌都遵循 **“条件：效果”** 的设计语法。
* **基础属性**：名称、占格形状 (Shape)、指示方向 (Direction)、基础 San 消耗。
* **标签系统 (Tags)**：卡牌之间联动的桥梁。基础标签包括：废弃物、机械、梦之种、食物、书籍、水、日常、珍稀、工具、玻璃、神秘。
* **状态标签**：如“污染”，为动态附着状态。

## 二、 三大核心流派 (Archetypes)

### 流派 A：污染流 (Corruption) - 极致的高风险高回报
* **核心机制**：物品可附着“污染层数 (n)”。
* **收益与代价**：带有污染的物品在被“撞击”结算时，其基础效果会额外触发 n 次，但同时会额外扣除 n 点 San 值（梦值）。每次游戏结束后污染层数清空。
* **玩法循环**：前期快速叠加污染层数，利用乘法机制获取爆炸分数；后期寻找“净化”手段消除污染，换取存活。

### 流派 B：机械流 (Machinery) - 空间与路径的精密规划
* **核心机制**：利用特殊形状和多向传导机制，改变撞击路径，实现一击多响。
* **特点**：极度依赖齿轮等转向件，以及全局控制件。目前处于设计初期。

### 流派 C：梦境流 (Dream) - 自动成长与形态转换
* **核心机制**：卡牌具有生命周期，满足条件后会发生形态替换（如：苹果 -> 苹果核 -> 苹果）。
* **特点**：长线投资，跨回合收益。目前处于设计初期。
