# 物品清单 - v1.3版本



## 一、修改原因

本版本补齐机械流派的第一版正式物品池。

机械流派的目标是围绕撞击结算、机械标签、曲线传动和命中数收益形成构筑。与污染流的层数成长、梦境之种的等级成长不同，机械流的主要成长来自玩家对背包空间的排布能力：能否让一次撞击结算命中更多机械物品，能否通过垂直方向的曲线传动延长路径，能否在本次撞击结算全部停止后获得高额收益。

机制细节先写入临时设计文档：

`../temp/Mechanical_Transmission_Temp_Design.md`



## 二、新增机制摘要

本版本建议机械流使用以下关键词：

| 关键词 | 含义 |
| --- | --- |
| 机械撞击！ | 只会命中机械物品的特殊撞击。若目标方向紧贴非机械物品，则该分支停止。 |
| 向左传动！ | 从当前物品的左侧方向发起一次机械撞击。左侧相对于当前物品朝向计算。 |
| 向右传动！ | 从当前物品的右侧方向发起一次机械撞击。右侧相对于当前物品朝向计算。 |
| 双向传动！ | 向自身方向的两个垂直方向各发起一次机械撞击。 |
| 全向传动！ | 向所有相邻机械物品各发起一次机械撞击。仅用于后期或限次物品。 |

为避免无限循环，机械传动采用以下边界：

1. 同一次撞击结算中，同一个物品最多被命中 1 次。
2. 重复指向已经命中过的物品时，该分支停止。
3. 生成传动分支的效果默认每个物品每次撞击结算最多触发 1 次。
4. `本次撞击结算全部停止后` 的效果每次撞击结算只结算 1 次。
5. 机械撞击不能穿过非机械物品。
6. 污染可以复制收益段，但不复制机械撞击、曲线传动等生成分支的传动段。

## 五、设计边界

1. 前期机械物品只提供小额分数、普通传导和基础转向，避免玩家过早形成高收益高命中结算。
2. 核心机械物品围绕机械命中数、转向传动次数、双向传动和结算停止后的奖励展开。
3. 后期补强要求较高梦值消耗和较大占格，收益依赖一次撞击结算中的高命中数。
4. `星环轴承` 使用 `每个局内关卡第一次被撞` 限制，避免全向传动反复制造分支。
5. `永动轮` 等会再次触发根源撞击的候选物品暂不加入，因为它们更容易造成循环和结算边界不清。
