# 临时设计 - 道具



## 一、基本定位

道具是一类特殊物品，但它不参与常规背包构筑结算。道具更接近“一次性消耗品”，用于在局内或局外对某个物品、某个格子、某条撞击链或某个系统状态产生一次性影响。

道具设计目标：

- 提供一次性决策，而不是长期被动收益。
- 与物品、饰品和流派形成组合。
- 提供玩家主动干预局势的手段。
- 补足某些构筑的短板，例如启动、净化、播种、换方向、临时补梦值。



## 二、道具基本规则

1. 道具属于物品的一种，但不放入普通背包格参与撞击结算。
2. 道具有独立的道具栏或堆叠显示。
3. 同一种道具可以在同一个格子中堆叠多个。
4. 玩家可以拖动道具到目标上使用。
5. 道具使用后数量 -1。
6. 如果目标不合法，道具不会消耗，并回到原位。
7. 道具可以通过商店、事件、关卡奖励、饰品效果或物品效果获得。
8. 道具原则上不应直接提供过高分数，而应改变物品状态或创造一次联动机会。



## 三、道具使用目标

道具可以有不同目标类型：

| 目标类型 | 说明 | 例子 |
| --- | --- | --- |
| 物品 | 拖到某个物品上生效 | 给物品+污染、净化、旋转、临时加分 |
| 空格 | 拖到背包空格上生效 | 播种、生成衍生物品、放置临时障碍 |
| 背包区域 | 拖到背包中任意位置生效 | 全局改变方向、触发一次扫描 |
| 饰品 | 拖到饰品上生效 | 暂时强化某个饰品、刷新饰品触发次数 |
| 捕梦区 | 拖到捕梦区生效 | 下一次捕梦改变概率或降低梦值消耗 |
| 垃圾桶/丢弃区 | 拖到丢弃区生效 | 强化下一次丢弃收益 |

## 五、道具与饰品的联动方向

### 1. 刷新饰品次数

`空白符纸` 可以刷新一个“每局限1次”的饰品。这类效果要非常谨慎，不能刷新会导致无限循环或长期收益过高的饰品。

适合联动：

- 收获篮
- 反冲片
- 空梦奖杯
- 密封瓶

不适合联动：

- 会直接产生大量梦值的饰品。
- 会生成大量道具的饰品。
- 会再次刷新饰品的饰品。

### 2. 道具生成饰品收益条件

部分饰品可以围绕“使用道具”触发。例如后续可以设计：

- 每使用3个道具：+10分。
- 使用道具到废弃物上时：该废弃物+1污染。
- 使用道具到梦境之种上时：额外+2分。

这类饰品可以让道具成为一个小流派，但不宜过多，否则会抢走背包构筑的主体位置。

### 3. 道具作为流派补洞

道具不应替代流派核心，而应补足临时短板：

- 污染流缺层数时，用黑墨滴、腐蚀酸。
- 梦种流缺成长时，用小水滴、肥料包、快速发芽剂。
- 机械流断链时，用链条钩、传动油、超频齿轮。
- 丢弃流缺收益时，用回收夹、苹果蜡。



## 六、后续筛选建议

正式进入版本前，道具池建议先控制在 15 个左右：

- 3 个通用道具。
- 3 个污染道具。
- 3 个梦境之种道具。
- 3 个机械道具。
- 2 个丢弃/食物道具。
- 1 个饰品联动道具。

当前这份 30 个候选用于临时筛选，不代表最终全部加入游戏。
