# 临时设计 - 机械曲线传动



## 一、设计目标

机械流派的核心不是单个物品高分，而是让玩家通过方向、贴合、转向和机械标签构筑更长的撞击传动路径。机械流派应当满足：

1. 前期能用少量机械物品获得稳定小分。
2. 中期通过向左传动、向右传动改变撞击路径，形成比普通同方向传导更长的结算。
3. 后期用命中物品数、命中机械物品数和转向传动次数作为主要收益。
4. 每次撞击结算必须有明确的终止规则，不能形成无限循环。



## 二、核心关键词

### 1. 本次撞击结算

机械流不再使用旧版抽象概念作为正式规则词。正式规则统一使用 `本次撞击结算`。

本次撞击结算指：

> 从一个撞击源发起撞击开始，到这次撞击产生的所有普通传导、机械传动和分支都停止为止。

程序实现时，可以把它视为一个临时结算上下文。这个上下文至少记录：

- 已命中的物品集合。
- 本次命中物品数。
- 本次命中机械物品数。
- 本次转向传动次数。
- 待结算的普通传导和机械传动分支。

如果同一个触发时机有多个物品发起撞击，则按顺序分别创建多个本次撞击结算。

### 2. 机械撞击！

机械撞击是一类特殊撞击。

普通撞击会命中前方紧贴的任意物品，并按现有撞击规则继续传导。机械撞击只会命中带有 `机械` 标签的物品：

- 如果指定方向上紧贴的是机械物品，则命中该物品。
- 如果指定方向上紧贴的是非机械物品，则该机械撞击分支停止，非机械物品不会被命中。
- 如果指定方向上没有紧贴物品，则该机械撞击分支停止。

机械撞击命中机械物品后，仍然会触发该物品的 `被撞` 效果。

### 3. 向左传动与向右传动

曲线传动是机械流派的专属空间玩法。它指的是某个机械物品被撞后，不只沿原本方向传导，还会向自身左侧或右侧发起机械撞击。

为了让文本和实现都清楚，正式词条使用以下写法：

| 词条 | 含义 |
| --- | --- |
| 向左传动！ | 从当前物品的左侧方向发起一次机械撞击。左侧相对于当前物品朝向计算。 |
| 向右传动！ | 从当前物品的右侧方向发起一次机械撞击。右侧相对于当前物品朝向计算。 |
| 双向传动！ | 从当前物品的左侧方向和右侧方向各发起一次机械撞击。 |
| 全向传动！ | 从当前物品向所有相邻机械物品各发起一次机械撞击。仅用于后期或每局限次物品。 |

方向示例：

| 当前物品朝向 | 向左传动方向 | 向右传动方向 |
| --- | --- | --- |
| 向上 | 向左 | 向右 |
| 向右 | 向上 | 向下 |
| 向下 | 向右 | 向左 |
| 向左 | 向下 | 向上 |

### 4. 撞击统计

机械流派需要明确统计项，避免每个物品自己解释“长不长”。

| 统计项 | 定义 | 用途 |
| --- | --- | --- |
| 本次命中物品数 | 本次撞击结算命中过的不同物品数量，包含普通物品和机械物品。 | 通用长传动奖励。 |
| 本次命中机械物品数 | 本次撞击结算命中过的不同机械物品数量。 | 机械流主要收益。 |
| 本次转向传动次数 | 本次撞击结算中，向左传动、向右传动、双向传动或全向传动成功命中机械物品的次数。 | 曲线传动奖励。 |
| 成功传导 | 某个被撞物品让后续至少 1 个物品被命中。 | 判断传动件是否发挥作用。 |

`本次撞击结算全部停止后` 指的是当前撞击源以及它产生的所有分支都结算完成。



## 三、结算顺序

建议按以下队列模型结算：

1. 某个物品发起一次撞击，创建一次本次撞击结算。
2. 当前分支命中物品后，先触发该物品的 `被撞` 效果。
3. 如果 `被撞` 效果生成机械撞击分支，则把新分支加入本次撞击结算的待结算队列。
4. 当前分支按普通撞击规则尝试继续传导。
5. 队列中所有分支结算完成后，触发所有 `本次撞击结算全部停止后` 效果。

如果同一时间有多个分支待结算，建议按以下顺序：

1. 生成分支的物品更靠近左上角者先结算。
2. 同一物品生成多个方向时，按上、右、下、左顺序结算。
3. 同一条件仍然相同，则按生成顺序结算。



## 四、防无限循环规则

机械流派必须明确防止环路刷分。本版本建议采用以下规则：

1. 同一次撞击结算中，同一个物品最多被命中 1 次。
2. 如果某个分支即将命中本次撞击结算中已经命中过的物品，该分支立刻停止，不触发 `被撞`，不计入命中数。
3. 生成传动分支的效果默认每个物品每次撞击结算最多触发 1 次，除非词条明确写出更低频率，例如“每个局内关卡限1次”。
4. `本次撞击结算全部停止后` 效果在每次撞击结算中只结算 1 次。
5. 机械撞击不能穿过非机械物品，非机械物品可以自然截断机械分支。
6. 实现侧可以额外设置安全上限：每次撞击结算最多处理 49 个不同物品或 64 个分支步骤，超过时强制结束当前撞击结算。正常规则下不会触发该上限，它只是防止实现错误。

### 污染复制边界

污染会额外触发物品的结算收益，但不应复制“生成新撞击分支”的传动段，否则机械流会因为污染进入指数分支。

建议实现时将机械物品效果拆成两类：

- 收益段：加分、回复梦值等，可以被污染额外触发。
- 传动段：撞击、机械撞击、向左传动、向右传动、双向传动、全向传动，不被污染额外复制。



## 五、分数设计原则

机械流派的分数来源应当以命中数量和传动质量为主。

### 1. 前期

前期物品应该给稳定小分，让玩家愿意留下机械物品。

推荐范围：

- 单次被撞：+3 到 +8 分。
- 成功传导：额外 +2 到 +4 分。
- 本次命中 3 到 4 个机械物品：额外 +8 到 +12 分。

### 2. 中期

中期开始奖励机械命中数和转向传动次数。

推荐范围：

- 本次撞击结算全部停止后：+本次命中机械物品数 x 1 到 x2 分。
- 本次命中 6 个以上机械物品：额外 +10 到 +20 分。
- 本次转向传动 2 次以上：额外 +10 到 +20 分。

### 3. 后期

后期奖励已经成型的长传动路径，但需要高阈值和每次结算限次。

推荐范围：

- 本次命中机械物品数大于等于 10：+50 到 +70 分。
- 本次转向传动次数大于等于 3：额外 +20 到 +30 分。
- 高阶物品可以提供强分支能力，但应当每个局内关卡限 1 次。

## 七、正式入表建议

本版本建议正式写入 15 个机械物品：

| 类型 | 数量 | 物品 |
| --- | ---: | --- |
| 前期过渡 | 6 | 铁珠、小齿轮、传动皮带、左传动弯管、右传动弯管、制动片 |
| 核心物品 | 6 | 齿条、差速器、双轴转轮、蓄能飞轮、曲轴、计数轮 |
| 后期补强 | 3 | 中央引擎、终端计算机、星环轴承 |

这套池子的重点是：

- 先让玩家理解普通传导和小额收益。
- 再引入向左传动、向右传动和双向传动。
- 最后让命中机械物品数和转向传动次数成为后期分数核心。
