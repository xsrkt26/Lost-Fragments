# 物品



## 一、设计目的

物品是游戏构筑的核心组件，玩家需要使用抽出来的物品及商店中购买的物品于背包中进行构筑，达到目标分数。

## 三、物品词条设计

物品的各种词条基本遵循如下书写与设计逻辑：

“条件”  “：” “结算效果”

比如“每捕梦3次：撞击！”，每捕梦三次就是条件，撞击！是触发效果。



**词条定义：**

捕梦：指的是玩家在左侧用捕梦网捞取物品的操作。

撞击！：物品向它指示的方向发起撞击，撞击会在方向相同的物品上传导。

被撞：被撞是一个触发条件，当物体被撞击/被传导撞击后，会触发后面的内容。

梦值：玩家捕梦会消耗梦值，部分物品会额外扣除梦值/回复梦值。

污染：污染是附着在物品上的效果，每一次局内游戏结束后物品上的污染层数会被清空，下一次游戏开始时重新开始叠加。一个物品上具有n层污染，会让物品的结算效果额外触发n次。

净化：一般是清除物品上的污染层数，根据层数产生额外效果。

播种：如果所指方向第一格为空，则创造一个1级梦境之种。如果存在一个梦境之种，则使其等级+1。

衍生物品：带有衍生物品标签的物品在一局局内游戏结束后会从背包中消失。

## 五、物品操作方式

在游戏中，玩家拿到物品后，可以左键按住并拖动拖动至背包内或背包外。拖动至背包内，物品会具有网格吸附效果，同时只有背包内的物品才会参与结算。拖动至背包外可以放在任意位置，但是它们不会参与结算。

玩家鼠标放置在物品上会显示上文所示的描述页面，右键点击物品可以使其顺时针旋转（旋转后其可能和别的物品重合，或是部分位于背包外，此时它会自动视为在背包外，需要重新拖动并放入背包中），拖动物品时不会显示描述页面。



## 六、撞击的结算方式

撞击的结算逻辑如下：

1. 一般有一个物品发起撞击，这个物品称为撞击源。
2. 物品撞击会向它此时所朝的方向进行撞击，如果有另一个物品紧贴着当前物品并且**方向相同**，撞击会传导至下一个物品。
3. 如果一个物品所朝的方向没有紧贴物品，或者是一个物品所朝的方向紧贴的物品方向不同，那么撞击将不会继续传导。
4. 如果有多个物品发起撞击，那么谁更靠近左上角（相同距离的话，谁更靠近左侧）谁先发起撞击，此次撞击完全结算后，再开始结算后续的撞击。

每个物品有一个方向，比如上图中，3号物品方向向下，其余所有物品方向都是向左。我们假定1号物品是棒球，一个1格大小的物品，效果为“当你捕梦到另一个棒球时，撞击！”
